# 🎬 Netflix Movies Analysis

## 📌 About This Project
This project performs Exploratory Data Analysis (EDA) 
on a Netflix movies dataset. The goal is to find 
interesting trends and patterns in Netflix movies 
like which genres are most common, which movies are 
most popular, and how releases have grown over the years.

---

## 📂 Dataset Information
- **File:** `mymoviedb.csv`  
- **Total Movies:** 9827
- 
---

## ❓ Questions Answered
- 🎭 What is the most frequent genre on Netflix?
- ⭐ How are movies distributed by rating category?
- 📅 Which year had the most movie releases?
- 🏆 Which movie got the highest popularity score?
- 📉 Which movie got the lowest popularity score?
- 🎯 Which genre has the highest average popularity?
- 🔝 What are the Top 10 most popular movies?

---

## 🛠️ Libraries Used
- **Pandas** — data loading, cleaning & processing
- **NumPy** — numerical calculations
- **Matplotlib** — basic plots & charts
- **Seaborn** — styled visualizations

---

## 📊 Key Findings
- **Most Popular Movie:** Spider-Man: No Way Home
- **Most Common Genre:** Drama
- **Peak Release Year:** 2020
- **Dataset has no duplicate values**
- **Vote categories:** Popular, Average, 
   Below Average, Not Popular
